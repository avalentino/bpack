Installation
============

Pip
---

.. code-block:: shell

  $ python3 -m pip install bpack


Conda
-----

.. code-block:: shell

  $ conda install bpack


Testing
-------

To run the test suite it is necessary to have pytest_ installed:

.. code-block:: shell

  $ python3 -m pytest --pyargs bpack

.. _pytest: https://docs.pytest.org
